# 交接班摘要测试集

本目录用于评估护理场景下大模型从口头交班、护理记录或病情片段中生成结构化交接班摘要的能力。测试样例覆盖术后护理、急诊留观、慢病急性加重、儿科、产科、老年、肿瘤、神经内科和肾内科等情境。

## 文件

- `dataset.jsonl`：测试数据，每行一个 JSON 对象。
- `prompt.md`：该场景推荐使用的系统提示词，要求模型只输出 JSON。

## 数据集字段

- `id`：样例编号。
- `scenario`：护理交接场景。
- `input`：输入给模型的原始交班内容，模拟临床口头交班或护理记录片段。
- `expected_output`：优秀结果示例，采用 SBAR 结构，并包含风险、管路、待办和升级事项。

## 评判重点

- 是否完整提取 Situation、Background、Assessment、Recommendation。
- 是否保留关键数值，如生命体征、血糖、尿量、引流量、时间点、药物剂量和氧流量。
- 是否识别高风险护理问题，如出血、感染、误吸、跌倒、低氧、过敏、谵妄、血糖异常、导管相关风险。
- 是否准确列出管路和设备状态，并给出对应护理要点。
- 是否明确下一班待办事项和需要通知医生或升级处理的触发条件。
- 是否避免编造未提供的信息，且输出为可解析的 JSON。
