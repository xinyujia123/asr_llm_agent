# 慢病随访测试集

本目录用于评估护理大模型在“慢病随访”场景中的结构化判断与护理建议能力。样例覆盖糖尿病、高血压、COPD、慢性心衰、慢性肾病、腹膜透析及多病共存等常见社区和居家慢病管理情境。

## 评判重点

- 能否准确提炼病情摘要，区分稳定问题、控制不佳和急性加重风险。
- 能否评估用药、饮食运动、复诊检查、透析或吸入装置等依从性问题。
- 能否给出符合疾病特点的居家监测项目、频率、目标和记录方式。
- 能否进行合理风险分层，并用患者信息支撑判断。
- 能否提出可执行的护理随访建议，而不是替代医生处方。
- 能否识别需就医或升级处理的红旗症状。
- 能否提供专业、清晰、适合患者和家属理解的健康教育。

## 数据集字段

`dataset.jsonl` 每行是一个 JSON 对象，字段包括：

- `id`：样例唯一编号。
- `scenario`：测试样例所属慢病随访子场景。
- `input`：患者或家属提供的随访原始信息。
- `expected_output`：优秀参考输出，包含：
  - `病情摘要`
  - `依从性`
  - `居家监测`
  - `风险分层`
  - `随访建议`
  - `需就医/升级条件`
  - `健康教育`

`prompt.md` 提供该场景的系统提示词，要求模型只输出结构化 JSON，并覆盖上述必要字段。
